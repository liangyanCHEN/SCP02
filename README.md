# **Introduce**

Example for **GP SCP02** based on Android, test on Android 12

# **Usage**

Put below dependency(SCP03 ONLY)

`implementation 'org.bouncycastle:bcpkix-jdk15on:1.70'`

Use **Initial update** and **external authentication**

`initUpdate`

`extAuthCmd`
